# CrewAI Agentic Projects

Welcome to the CrewAI projects, powered by [crewAI](https://crewai.com).   

## Installation

Ensure you have Python >=3.10 <3.14 installed on your system.   
This project uses [UV](https://docs.astral.sh/uv/) for dependency management and package handling, offering a seamless setup and execution experience.   

First, if you haven't already, install uv:

```bash
pip install uv
```

## Running the Project

**Add your `KEYS` into the `.env` file**

To kickstart crew and begin task execution, run this from the root folder of your project:

```bash
$ cd agent
$ crewai run
```
